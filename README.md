CloudKidTasks
=============

Asynchronous task manager for CloudKidOS

##Documentation##

http://cloudkidstudio.github.io/CloudKidTasks/

##Installation##

CloudKid Tasks can be install using Bower.

```bash
bower install cloudkid-tasks
```
